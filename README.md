## :robot: Password Generation App

A REST API written in [Node.js](https://nodejs.org/en) wth [TypeScript](https://www.typescriptlang.org/) deployed to [AWS Elastic Beanstalk](https://aws.amazon.com/elasticbeanstalk/)
via [AWS CodePipeline](https://aws.amazon.com/codepipeline/).

A [React](https://react.dev) app written in [TypeScript](https://www.typescriptlang.org/) deployed to [Vercel](https://www.vercel.com).
Uses [pwbot-api](https://github.com/jgnovakdev/pwbot-api) as the back-end service for generating passwords.

## :paintbrush: Style

A little experiment with <a href="https://dribbble.com/tags/neo%20brutalism" target="_blank">Neo Brutalism</a>.
