## :robot: Password Generation App

A <a href="https://react.dev" target="_blank">React</a> app written in <a href="https://www.typescriptlang.org/" target="_blank">TypeScript</a>
deployed to <a href="https://www.vercel.com" target="_blank">Vercel</a> that uses <a href="https://github.com/jgnovakdev/pwbot-api" target="_blank">pwbot-api</a>
as the back-end service for generating passwords.
